Library Manager
Модульная система управления библиотекой.

Описание

Library Manager - учебный проект для изучения принципов модульного
программирования. Система позволяет управлять каталогом книг библиотеки.

Функциональность

- Добавление книг в каталог
- Просмотр списка книг
- Поиск книг по различным критериям
- Выдача книг читателям
- Возврат книг
- Статистика библиотеки

Модульная архитектура

Проект разделен на следующие модули:

Слой данных (Models/)

- Book.cs — класс книги
- Reader.cs — класс читателя
- Loan.cs — класс выдачи

Слой бизнес-логики (Business/)

- LibraryCore.cs — основная логика библиотеки
- SearchService.cs — поиск книг
- StatisticsService.cs — статистика и отчёты

Слой интерфейса (Interface/ + Views/)

- MainWindow.xaml/.cs — главное окно
- AddBookWindow.xaml/.cs — окно добавления книги

Вспомогательный слой (Utils/)

- ValidationHelper.cs — валидация данных
- FormatterHelper.cs — форматирование вывода
